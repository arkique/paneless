Paneless is a simple, clean WordPress theme for internal network use with user menu and integration placeholders.

== Description ==
A classic PHP-based WordPress theme designed for internal network dashboards. Features include:

* Dark/Light mode following system preference
* User menu (login/logout/profile dropdown)
* Integration widget area for Sonarr/Radarr API placeholders
* Dashboard cards showing recent activity
* WooCommerce compatible

== Installation ==
1. Download the theme folder
2. Upload to wp-content/themes/
3. Activate in WordPress Admin
