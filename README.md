# Paneless

A simple, clean WordPress theme for internal network use.

## Installation

Copy the `paneless/` folder to your WordPress `wp-content/themes/` directory and activate in WP Admin.

## Features

- **Dark/Light Mode** - Automatically follows system preference (`prefers-color-scheme`)
- **User Menu** - Shows login link or user dropdown with profile/logout options
- **Integration Area** - Widget area ready for Sonarr/Radarr API integrations
- **Dashboard Cards** - Quick overview of recent activity and links
- **Customizer Options** - Accent color and copyright text settings
- **WooCommerce Friendly** - Declares WooCommerce support with product gallery features (zoom, lightbox, slider)
